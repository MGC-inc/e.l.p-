# タスク.md — 横断タスク管理

> 議事録・通話ログ・日常業務から発生したTODOを一元集約する場所。
> 「未完了タスクを一覧にして」はこのファイルだけ見れば答えられる状態を保つ。

## 3. 運用ルール

1. **議事録・通話要約でTODOが発生したら、このファイルにも転記する**（発生元のファイル名を必ず書く）
2. タスクは「誰が・いつまでに・何を」の形で書く
3. 完了したら「2. 完了タスク」へ移動（削除しない — 月末整理まで残す）
4. 週次でこのファイルを見直し、期限切れ・放置タスクを洗い出す
5. KPIに直結するタスクは [KPI.md](./KPI.md) のHowとも整合させる
