# 訪問営業 業務マニュアル

- **対象業務**: 戸建て個人宅への訪問営業（エコキュート・エコジョーズの提案・販売）
- **担当**: 営業担当（アポインター）／クローザー
- **頻度**: 毎営業日
- **最終更新**: 2026-06-10

---

## 1. この業務の目的・完了条件

- **目的**: お客様が気づいていない設備・水質の潜在的な問題に気づいていただき、解決策を提示する。「良い人が来てくれた」と感じていただける関わり方をすること
- **完了条件**: 契約書・必要書類一式の取得、TimeTreeへの工事日程登録、入金確認

---

## 2. 全体の流れ

```
① エリア指定（管理者）
      ↓
② 飛び込みアポイント取得（アポインター）
      ↓
③ アポ取得時の当日対応（LINE登録・TimeTree登録・挨拶LINEメッセージ）
      ↓
④ 商談前フォロー（準備物の案内LINEメッセージ）
      ↓
⑤ 商談・クロージング（クローザー ※主権者2名在宅が必須条件）
      ↓
⑥ 契約書類の取得
      ↓
⑦ 工事手配・完工
      ↓
⑧ 入金（完工後2週間以内目安）
```

---

## 3. 各ステップの詳細手順

### ステップ①：エリア指定

- 管理者がエリアを指定する。市場全体を見ながら決定
- **クローザーの商談時間・移動時間を考慮**したエリア設定が重要。クローザーが間に合わないエリアは避ける
- 担当エリアに変更がある場合は管理者に確認する

---

### ステップ②：飛び込みアポイント取得

**切り口（最初の一言）**
- ハウスクリーニングのご案内
- エコキュートのメンテナンスのご案内

**アポイントの条件（必須）**
- **主権者2名（ご夫婦など）のご在宅時に商談の場を設定する**
- 単身のお客様の場合はこの限りではない
- 1名しか対応できない場合は、サービス提供自体ができないためリスケを促し、両名揃う日程で再設定する

> **なぜ2名在宅が必須か**
> 商談は即決を前提としている。「確認してから決める」「保留」は基本的に失注につながるため、その場で決断いただける状況をアポイント段階で整える必要がある。

---

### ステップ③：アポ取得時の当日対応（アポインター）

| 対応 | 方法 | タイミング |
|------|------|----------|
| 公式LINEへの登録案内 | QRコードを提示してその場で登録していただく | アポ取得直後 |
| 挨拶・お礼LINEの送信 | 公式LINEから送信 | 登録確認後すぐ |
| TimeTreeへの商談日程登録 | アポインター本人が共有カレンダーに入力 | アポ取得当日中 |

---

### ステップ④：商談前フォロー（アポインター）

商談当日の前日〜当日朝にLINEで以下を案内する。

**お客様に準備いただくこと**
- 最低2時間、話を聞くスタンスでお時間をいただく旨をあらためてご確認
- 追い炊き配管の清掃を希望される場合は**浴槽にお湯を張っておいていただく**（この場合のみ）

---

### ステップ⑤：商談・クロージング（クローザー）

#### 5-1. 持参物チェックリスト

| 品目 | 用途 |
|------|------|
| ブック（資料ファイル） | 順に見せながら説明するコピー資料を格納 |
| 白紙の紙（複数枚） | 手書きで説明し、最後お客様にお渡しする |
| 見積書 | 金額提示 |
| 売買契約書 | 契約締結 |
| 保証会員証 | 保証内容の説明・お渡し |
| 写真使用同意書（任意） | 施工事例等への使用許可 |
| メンテナンスチェックシート | 家の設備情報を記入いただく（契約時） |
| スマホ | 施工写真・事例写真の提示 |
| クリアなコップ | 塩素濃度実験用 |
| 試薬 | 水道水の塩素濃度測定 |
| ドライバー等工具 | エコキュートメンテナンス時のみ |

#### 5-2. 商談の進め方（ブックの順番）

```
① 水質の説明
      ↓
② 弊社の保証サービスの内容
      ↓
③ 解決策・商品紹介（浄水器の説明）
      ↓
④ 金額提示（見積書を提示）
      ↓
⑤ クロージング
```

#### 5-3. 実験（塩素濃度測定）のタイミングと目的

- **タイミング**: ③商品紹介のフェーズで実施
- **目的**: コップに水道水を入れ、試薬を加えて塩素濃度を可視化する。現状の水道水の状態をお客様自身の目で確認してもらうため

#### 5-4. クロージングの原則

- **その場での即決**を目指す
- 「保留・後日確認」は基本的に失注と同義と心得る
- 迷っているお客様に対しては、保証内容・サービス価値を丁寧に再説明する

---

### ステップ⑥：契約書類の取得

契約締結時に以下を一式取得・記入いただく。

- 売買契約書
- メンテナンスチェックシート（家の設備情報・わかる範囲で記入）
- 写真使用同意書（任意）
- 保証会員証（お客様用を渡す）

---

### ステップ⑦：工事手配・完工

- 契約後、工事の日程調整・業者手配を行う
- **銀行ローン契約の場合**: 審査通過後に工事（契約後1〜2週間目安）

---

### ステップ⑧：入金

| 支払い方法 | 入金タイミング |
|-----------|--------------|
| 現金・振込 | 完工後2週間以内目安 |
| 銀行ローン | 審査通過 → 工事 → 銀行から直接 or お客様経由で振込（銀行により1〜2週間の差あり） |

## 4. 品質基準・チェックリスト

### アポイント時
- [ ] 主権者2名の在宅日程で設定できているか
- [ ] 公式LINEを登録してもらえたか
- [ ] TimeTreeに日程を入力したか
- [ ] 挨拶LINEを送ったか

### 商談前
- [ ] 商談前日〜当日朝にLINEで準備物の案内を送ったか
- [ ] 持参物に不足がないか（チェックリスト参照）

### 商談・クロージング時
- [ ] 主権者2名が在宅しているか（1名の場合はリスケ）
- [ ] ブックを順番通りに進めているか
- [ ] 実験を③のタイミングで行ったか
- [ ] 契約書類を一式取得できたか

## 5. マインドセットと危機管理

### マインドセット

> お客様が自分では気づけない潜在的な問題に気づいていただき、解決策を提示してあげられる「良い人」であるというスタンスを持つ。

- 訪問営業は「押し売り」ではなく「問題解決の提案」
- お客様の生活を良くするために動いているという自覚を持つ
- 断られても、それはサービスが不要だったということ。人格否定ではない

### 危機管理（してはいけないこと）

- スタンスを履き違えて言い争いに発展させること
- お客様に混乱・迷惑・不安をもたらすこと
- 自分の影響力（訪問という行為のもつ力）を軽視すること

> 営業担当者の言動はお客様の生活に直接影響する。「熱心さ」と「強引さ」は別物と心得ること。
