# 目的
アップロードされた電話録音の音声ファイルを分析し、「文字起こし」「内容要約」「指定のヘッダー順（全19項目）に完全に一致するスプレッドシート1行貼り付け用（タブ区切り）データ」を作成してください。

# 処理フロー
1. 会話内容を全文文字起こしする。
2. 会話内容を3行以内で簡潔に要約する。
3. 指定の抽出ルールに従って各項目を抽出する。
4. スプレッドシートのA列〜S列（全19項目）に過不足なく一発でペーストできる「タブ区切りテキスト」を作成する。

# 抽出項目と選択肢のルール
- 案件業者: 「車トラブル受付センター」か「車トラブル相談窓口」
- 問い合せ日: YYYY/MM/DD形式
- 問合せ時間: HH:MM形式
- エリア: 愛知県 / 岐阜県 / 三重県 / その他(県名)
- 住所①: 市区町村まで
- 住所②: 市区町村以下の番地・建物名など
- お客様名: 氏名を抽出
- 性別: 男性 / 女性 / 不明
- お客様電話番号: ファイル名や会話内から抽出
- 電話結果: 成約 / 保留 / NG / NG(kick) / 応答なし / 営業電話
- NG、保留理由: 成約 / 対応不可 / JAF間違え / 間違い電話(JAF除く) / 他社に決まった / 料金確認のみ / 金額聞いてキャンセル / 時間遅くてキャンセル / 自己解決 / 非対応地域 / 修理方法の質問 / その他
- 車両種別: 普通車 / バイク / トラック / その他
- トラブル種別: バッテリーあがり / パンク / エンジントラブル / 事故 / ガソリン切れ / 脱輪 / 雪道スタック / 鍵閉じ込み / オイル漏れ / その他
- 保険会社名: ソニー損害保険 / SBI損害保険 / アクサ損害保険 / 三井ダイレクト損害保険 / チューリッヒ保険会社 / 東京海上グループ / SOMPOグループ / 三井住友海上火災保険 / あいおいニッセイ同和損害保険 / 共栄火災海上保険 / こくみん共済 coop / その他 / なし

【条件付き項目】※該当しない場合は必ず「空欄（タブのみ）」にする
- 「時間遅くてキャンセル」の場合のみ → 何時間と伝えたかを抽出
- 「自己解決」の場合のみ → どうやって解決したかを抽出
- 「非対応地域」の場合のみ → どこの地域かを抽出
- 「金額聞いてキャンセル」または「他社に決まった（キャンセル料発生など）」場合 → 回答金額を抽出

# スプレッドシート配置ルール（厳守）
出力するタブ区切りテキストは、必ず以下の全19項目（A列〜S列）の順番と一致させてください。値がない項目は、タブのみを入れて列を飛ばしてください。

A列: タイムスタンプ（※データ生成時点の現在日時「YYYY/MM/DD HH:MM:SS」を自動挿入）
B列: 案件業者
C列: 問い合せ日
D列: 問合せ時間
E列: エリア
F列: 住所①
G列: 住所②
H列: お客様名
I列: 性別
J列: お客様電話番号
K列: 電話結果
L列: NG、保留理由
M列: 車両種別
N列: トラブル種別
O列: 保険会社名
P列: 何時間と伝えたか（該当日時以外は空欄）
Q列: どうやって？（該当日時以外は空欄）
R列: どこ？（該当日時以外は空欄）
S列: 回答金額（該当日時以外は空欄）

# 出力フォーマット
以下の構成で出力してください。

---
【文字起こし】
（ここに全文の文字起こし）

【要約】
（ここに3行以内の要約）

【スプレッドシート1行貼り付け用】
※以下のコードブロック内のテキストをコピーしてA列に貼り付けると、そのままS列まで綺麗に展開します。

```text
[ここにタイムスタンプから回答金額までをタブで区切った1行の文字列を出力]
```
