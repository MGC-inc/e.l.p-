# 通話ログ/ — イノベラ通話要約の保存先

> イノベラ（電話録音）の要約を蓄積するフォルダ。LINE通知だけだと流れて消えるため、ここに残して「先週の電話の内容は？」に答えられる資産にする。

## 1. ルール

- **保存場所**: `通話ログ/YYYY/`（年ごとのフォルダ）
- **ファイル名**: `YYYY-MM-DD_相手.md`
  - 例: `2026-06-10_〇〇ビル管理.md`
  - 相手の表記は [../顧客一覧.md](../顧客一覧.md) に合わせる
  - 同日に同じ相手と複数回通話した場合は `YYYY-MM-DD_相手_2.md` のように連番
- **内容**: [../イノベラ.md](../イノベラ.md) の要約テンプレートをそのまま使う
- LINE通知と同時にここへもコミットする（`docs(通話ログ): 2026-06-10 〇〇社` 形式）

## 2. 議事録との使い分け

| | 通話ログ/ | 議事録/ |
|---|---|---|
| 対象 | 電話のやり取り | MTG（対面・オンライン） |
| 粒度 | 要約テンプレのまま | 決定事項・TODO中心 |
| 昇格 | 重要案件は議事録に転記（元の通話日時を明記） | — |

## 3. Claude Code への指示例

- 「先週の〇〇社との電話の内容を教えて」
- 「今月の通話ログから対応が必要なものを抜き出してタスク.mdに転記して」
