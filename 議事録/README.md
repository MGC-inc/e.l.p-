# 議事録/ — E.L.P における議事録の作り方

## 1. ルール

- **保存場所**: `議事録/YYYY/` （年ごとのフォルダ）
- **ファイル名**: `YYYY-MM-DD_相手_テーマ.md`
  - 例: `2026-06-10_東さん_イノベラ運用MTG.md`
- **作成タイミング**: MTG当日中にコミットする
- **テンプレート**: [_template.md](./_template.md) をコピーして使う

## 2. 書き方の方針

- 発言の逐語録ではなく **決定事項・TODO・論点** を中心に書く
- TODOは必ず「誰が・いつまでに・何を」の形にする
- KPIに関わる決定は [../KPI.md](../KPI.md) にも反映する
- イノベラの通話要約から昇格した議事録は、元の通話日時を明記する

## 3. Claude Code への指示例

- 「東さんとの直近のMTG議事録を踏まえて、次回アジェンダを作って」
- 「6月の議事録からTODO未完了のものを一覧にして」
